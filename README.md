## Preview

One Line is a minimal anonymous web experience built using Flask.

Visitors can submit a single thought anonymously while only the admin can privately access the archive.

The project was designed to feel:
- cinematic
- emotionally quiet
- minimal
- intimate

---

## Features

- Anonymous submissions
- Private admin dashboard
- Beautiful dark UI
- Responsive design
- Smooth animations
- SQLite database
- Flask backend
- Session-based admin authentication

## Philosophy

Most social platforms encourage noise.

One Line was built around restraint.

A single sentence.
No usernames.
No likes.
No algorithms.

Just fleeting thoughts left behind anonymously.

---

## Local Installation

Clone repository:

```bash
git clone https://github.com/Moksh-dev/one-line.git
```

Enter project folder:

```bash
cd one-line
```

Install Flask:

```bash
pip install flask
```

Run app:

```bash
python app.py
```
